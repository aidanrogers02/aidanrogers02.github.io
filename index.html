Projects

Taylor Swift Spotify Data Analysis

A project analyzing the Spotify data of Taylor Swift

A personal project I wanted to work on to find statistical trends in Taylor Swift's music and to demonstrate my skills in data analysis. I worked with datasets in Kaggle that have quantitative Spotify data on Taylor Swift's music. I specifically wanted to see how the different sonic qualities present in Taylor Swift's albums impact their popularity. I used Excel, R, SQL, and Tableau to analysis and present this data. I also used techniques such as descriptive statistics and linear regression.
1. Full analysis and code: GitHub repository
2. Analysis article on Medium
3. Interactive Dashboard: Tableau
4. Final presentation: PowerPoint
https://github.com/aidanrogers02/Taylor-Swift-Spotify-Popularity-Analysis
Google Data Analytics Capstone: Cyclistic

A capstone data analysis project

My final capstone project for the Google Data Analytics Certification. I tackled a sample set of data from a fictional bike sharing and went through all of the steps of the data analysis process with it: ask, prepare, process, analyze, share, and act. Album Popularity Trends

Graph the popularity of an artist's albums

This Python project takes the input of the user and displays a plotly scatter chart that shows the popularity of an artist's albums over the years. This popularity is determined and attained via the Spotify API. This project heavily utilizes the Spotify API to get an artist's page, albums, and album stats. Initially this project was made to view the popularity of Mariah Carey's official album releases so her results are the most tailored.

Card Game Analysis

Play Go Fish and see stats on the results

A Python project where you can play go fish and then have the results of the game(s) be graphed out for you with plotly. The Go Fish game is made using Objected Oriented Programming in making classes for the cards, card decks, and players. The player can play go fish themselves or have the computer play itself 1 or 100 times to get a solid dataset to graph.
